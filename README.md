# Role-Based Project Management Backend

## [Project Management Backend Live Url](https://project-management-server-woad.vercel.app)

## Description

This is the backend for a Role-Based Admin & Project Management System with **invite-only user registration**. Admins manage users and projects, while users can participate in project creation.

Key features:

- Invite-based user registration (no self-sign-up)
- Role-Based Access Control (ADMIN, MANAGER, STAFF)
- JWT-based authentication
- Soft delete for projects
- Admin user management (role & status updates)
- Centralized validation & error handling
- TypeScript with clean architecture

## Tech Stack

- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT for authentication
- bcryptjs for password hashing
- dotenv for environment configuration
